Steps to load dataset in your MySQL Server.
STEP 1:	Paste the download file 'dataset.sql'in any location
STEP 2: In the same location, launch MySQL Client with your credentails
STEP 3: Type the command 'SOURCE dataset.sql;' (Database and the tables will be created)
STEP 4: Type the command 'SHOW TABLES;', to view the created tables required for demos and exercises
